# Online Store Project

This project is a simple online store created with PHP. It allows users to browse products, add them to the cart, and place orders.

## How to Run

To run this project, follow these steps:

1. Clone this repository to your local machine.
2. Install PHP and a web server (e.g., Apache) if you haven't already.
3. Configure the web server to serve the project directory.
4. Import the database schema from the `database.sql` file into your MySQL database.
5. Update the database connection settings in the `config.php` file.
6. Navigate to the project directory in your web browser to access the application.

## Theme

The theme of this project is a generic online store, featuring various categories of products for sale.
